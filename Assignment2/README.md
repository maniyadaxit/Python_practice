# Task 1: Check if a Number is Even or Odd

```python
a = int(input("Enter the a number : "))

if a%2!=0:
print(f"{a} is odd number")
else:
print(f"{a} is even number")
```

# Task 2: Sum of Integers from 1 to 50 Using a Loop

```python

sum = 0
i=1

while i!=51:
  sum+=i
  i+=1

print(f"sum = {sum}")


```
